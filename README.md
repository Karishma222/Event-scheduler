Event Scheduler & Resource Allocation System

📌 Project Overview

The Event Scheduler & Resource Allocation System is a web application developed using Flask.
It helps users create and manage events, allocate resources, and avoid scheduling conflicts.
The system provides a simple interface to organize events efficiently and track resource usage.

---

🚀 Features

- Create, view, and delete events
- Allocate resources to events
- Prevent scheduling conflicts
- View event list with start and end time
- Resource utilization report

---

🛠 Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/your-username/event-scheduler.git

2. Navigate to the project folder
   cd event-scheduler

3. Install required packages
   pip install -r requirements.txt

4. Run the Flask application
   python app.py

5. Open in browser
   http://127.0.0.1:5000
